npm 发布中常见错误

  • 409 Conflict

npm login 或者 npm adduser 时 输入完账号密码邮箱后 提示 E409 Conflict 报错 一般因为淘宝镜像的原因

解决方案,直接 registry 镜像源

  • npm ERR! 403

可能是包名已经存在，或者是邮箱未认证
